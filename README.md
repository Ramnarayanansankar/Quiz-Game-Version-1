# Infinite Quiz Pro (v1.0)
### The Essential Endless Trivia Experience

**Infinite Quiz Pro** is a lightweight, high-performance web application designed for trivia enthusiasts. It leverages the Open Trivia Database API to provide a seamless, never-ending stream of questions across every imaginable topic.

## 🚀 Features
- **Endless Gameplay:** Questions are buffered in the background; the game only ends when you decide to close the tab.
- **Three Difficulty Tiers:** Choose between **Easy**, **Medium**, and **Hard** to match your expertise.
- **Lifelines:** - **50/50 Hint:** Removes two incorrect answers (with a small score penalty).
    - **Skip:** Move past a tricky question without losing points.
- **Glassmorphism UI:** A modern, semi-transparent interface designed for focus and aesthetic appeal.

## 🛠️ How to Run
1. Save the game code as `quiz.html`.
2. Open the file in any modern web browser (Chrome, Firefox, Edge, Safari).
3. **Note:** An active internet connection is required to fetch questions from the API.

## 💻 Technical Stack
- **HTML5/CSS3:** Custom animations and responsive layouts.
- **Vanilla JavaScript:** Asynchronous API handling and game logic.
- **API Source:** [Open Trivia DB](https://opentdb.com/)
